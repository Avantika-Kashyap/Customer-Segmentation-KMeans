# Customer Segmentation using K-Means Clustering

This project segments mall customers into distinct groups based on their *Annual Income* and *Spending Score* using the *K-Means Clustering* Machine Learning algorithm.

## 📊 Key Insights & Visualizations

### 1. Elbow Method (Finding Optimal Clusters)
We used the Elbow Method to find the best number of clusters, which turned out to be *5*.

### 2. Final Customer Clusters
The algorithm successfully segmented the customers into 5 distinct groups:
* *Target Customers:* High Income, High Spending (Primary focus for marketing)
* *Standard:* Average Income, Average Spending
* *Sensible:* High Income, Low Spending
* *Careful:* Low Income, Low Spending
* *Spendthrifts:* Low Income, High Spending
  
## 🛠️ Technologies Used
* Python
* Pandas & NumPy
* Scikit-Learn (KMeans)
